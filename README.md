# computing_project

Arcade style game, LEDS blink in circle pattern and user has to hit the button when the right LED (e.g. LED3) is on

Subroutines:
- delay_ms (time delay)
- blink 1 LED
- blink all LEDS in circle
- detect button press
- check if win or lose
- blink all LEDS if win, else blink target
